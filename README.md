# Projeto Amigo Secreto

## Descrição

Este projeto é uma aplicação web simples que permite realizar sorteios de amigos secretos. O usuário pode adicionar os nomes de seus amigos e, ao clicar no botão "Sortear", o sistema realiza o sorteio e exibe o nome do amigo sorteado.

## Funcionalidades

- **Adicionar amigos**: O usuário pode digitar os nomes dos amigos e adicionar à lista de participantes.
- **Sorteio**: Com pelo menos dois amigos na lista, o usuário pode clicar em "Sortear amigo" para realizar o sorteio.
- **Validação**: O sistema não permite adicionar nomes duplicados ou vazios.
